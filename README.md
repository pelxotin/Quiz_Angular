# 🎮 Quiz Angular

Projeto de **Quiz interativo** desenvolvido utilizando **Angular**.  
A aplicação apresenta perguntas e respostas onde o usuário pode testar seus conhecimentos de forma simples e interativa.

---

# 🚀 Tecnologias utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- Angular
- TypeScript
- HTML5
- CSS3
- Node.js

---

# 📚 Sobre o projeto

O objetivo deste projeto é praticar conceitos de **desenvolvimento frontend com Angular**, como:

- criação de componentes
- organização de estrutura de projeto
- manipulação de dados
- interação com o usuário
- lógica de programação aplicada a interfaces

Este projeto faz parte dos meus estudos em **desenvolvimento web e programação**.
